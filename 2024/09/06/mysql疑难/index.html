###Mysql的各种配置问题

####一、关于密码丢失（root用户忘记密码）

​ 丢失密码后最快速的方法就是重置密码。以下步骤作为参考。

  • 1、确认错误情况：

    这个错误就是表示配置方面没有问题，是密码的原因导致账户被拒绝访问。NO/YES表示你是这次操作有没有输入密码。

  • 2、跳过密码认证进入数据库操作：因为重置密码需要高权限，所以这里的操作都是在你本机可以操作root账户忘记密码用的。

    • 首先进入ProgramData目录下你的Mysql数据路径，这里放我自己的路径是C:\ProgramData\MySQL\MySQL Server 8.0

    • 然后在这个路径下用**管理员权限（必须）**打开命令控制板，在里windows系统用start命令打开my.ini配置文件

    • 找到[mysqld]这一行，在它下面任意一行输入**skip-grant-tables**

    • 这行命令用于跳过数据库登录时的密码验证。这时启动mysql，按照正常步骤登录进去。

      这里不用输密码，直接回车，后面跳出让你输入密码的对话框也直接回车，就可以登录进你的mysql里。

  • 3、重置密码：

    • 再输入这个命令(不要用UPDATE命令，那个命令需要输入的密码是加密后的字符串，这个命令就是接收普通字符串然后再加密存储。)

      之后刷新一下权限（我没刷新也行）。

      最后回到my.ini文件中将跳过密码验证的语句删掉，保存，重新启动mysql服务就完成密码的重置了。
